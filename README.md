# JS Email List

This is the 21st exercise I have completed as part of the web development master. It focuses on asynchronous JavaScript, fetching data from external APIs, and dynamically manipulating the DOM to display content.

Repository name: `js-email-list`

## 📝 Task

The objective of this assignment is to create a web application that generates a list of random email addresses using an external API.

The application must fulfill the following core requirements:
* Integrate the official Boolean API (`https://flynn.boolean.careers/exercises/api/random/mail`) to request random email entries.
* Perform a loop of asynchronous requests to successfully generate a total of 10 distinct email addresses.
* Dynamically render the retrieved email addresses inside an HTML list structure on the webpage once the data fetching process concludes.

### 🌟 Bonus
* **Styling:** Enhance the user interface and overall page layout using custom CSS or Bootstrap components.
* **Interactivity:** Add a dedicated interactive button that, when clicked, triggers a new fetch sequence to retrieve 10 fresh email addresses, completely replacing the previous list items.
